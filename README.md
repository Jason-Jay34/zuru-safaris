# Zuru Safaris

Zuru Safaris is a simple safari-themed website built using HTML and CSS.  
The project focuses on a clean layout, simple styling, and easy navigation.

## About the Project

This project represents a basic static website for a safari or travel concept.  
It is ideal for beginners learning how to structure web pages and apply CSS styling.

## Features

- Simple safari-themed homepage
- Image sections for destinations or safari views
- Easy-to-use navigation bar
- Basic responsive layout (if applied)
- Clean design using only HTML and CSS

## Technologies Used

- HTML5  
- CSS3

zuru-safaris/

## How to View the Project

1. Download or clone the repository.
2. Open the `index.html` file in any web browser.
